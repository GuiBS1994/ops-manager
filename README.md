# OPS Manager

Sistema integrado de gestão operacional e desenvolvimento pessoal, desenvolvido com Power Platform (Power Apps, SharePoint e Power Automate).

---

## Objetivo

Centralizar a gestão de atividades, pendências, projetos, indicadores e desenvolvimento pessoal em uma única aplicação, com foco em produtividade, organização e tomada de decisão.

---

## Arquitetura

* SharePoint Lists → Base de dados
* Power Apps → Interface do usuário
* Power Automate → Automações

---

## Funcionalidades

### 1. Atividades Recorrentes

* Cadastro de atividades fixas
* Frequência semanal
* Geração automática de agenda

### 2. Agenda Diária

* Execução das atividades
* Controle de status
* Postergamento automático

### 3. Pendências

* Gestão de demandas não recorrentes
* Integração com agenda

### 4. Fechamentos Mensais

* Controle mensal de atividades

### 5. Integração com E-mail

* Geração automática de tarefas
* Resumo de e-mails

### 6. Kanban

* Visualização das tarefas por status

### 7. Desenvolvimentos

* Gestão de projetos
* Controle de progresso e impedimentos

### 8. Manutenção

* Controle de moldes e equipamentos
* Geração de PDF

### 9. Indicadores

* Registro histórico de dados
* Consulta por período

### 10. Desenvolvimento Pessoal

* Pós-graduação
* Portfólio (GitHub)
* Ideias e insights
* Controle de inglês
